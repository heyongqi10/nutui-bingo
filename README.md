# NutUI-Bingo-React

React 版本的 NutUI-Bingo 组件库

Vue 版本：<https://github.com/jdf2e/nutui-bingo>

组件库模版：<https://github.com/xxhls/react-component-library-template>

## 特点

- 基于 React 18 + Typescript
- Monorepo 管理组件库及 Demo 示例
- 与 Vue 组件库完全匹配的 API
